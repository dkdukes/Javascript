# Simple ATM Simulation
## Check Balance
A user is able to check his/her balance
## Deposit 
A user provides the amount to deposit then the system returns
- Amount deposited
- Previous balance
- New balance
## Withdraw
A user provides the amount to deposit then the system checks if
- the user is verified
- the user has reached the daily limit
- the amount is in multiples of 100 <br/>
The system then responds with
- Amount withdrawn
- Fee applicable
- Remaining balance
